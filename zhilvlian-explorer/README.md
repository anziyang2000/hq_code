# exp
## 【新】区块链浏览器

### 准备环境

```bash 
yarn
```

or

```bash
npm install
```

### 启动项目

```bash
yarn start:prod  or  npm run start:prod // 项目没有dev环境和test环境的服务，所以需要启动正式环境
```

### 打包项目

```bash
yarn build  or  npm run build
```
